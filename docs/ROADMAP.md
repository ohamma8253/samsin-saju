## v1.0 — "신비로운 삼신과의 첫 만남" (핵심 경험 & 바이럴)

> 목표: 기존 콘텐츠의 극적 완성도 강화

### Must-Have

- [ ] **합의율 메타데이터 시스템**
  - `ConsensusLevel` 타입 추가 (unanimous / majority / conflict)
  - 3점수 기반 합의 강도 자동 계산 로직
  - AI 프롬프트에 합의 뉘앙스 규칙 추가 ("만장일치", "2:1 격론" 등)
  - FinalSynthesis에 `consensusLevel` 필드 추가

- [ ] **옥새 스탬프 애니메이션**
  - seal 표시 시 시각적 스탬프 찍히는 CSS 애니메이션
  - 합의 강도에 따라 금색(만장일치) / 은색(다수결) / 적색(충돌)
  - 화면 진동(shake) + 잔상(afterimage) 효과

- [ ] **삼신 합의 처방전(개운법)**
  - `Prescription` 타입: luckyColor, luckyDirection, luckyNumber, talisman
  - AI 프롬프트: "삼신이 합의하여 하사한 비책" 형식
  - 처방전 카드 UI (공유 최적화)

- [ ] **Progressive Disclosure UI**
  - 심층 분석 15섹션 → 아코디언(접기/펼치기)
  - 총운 리포트 → 핵심 요약 먼저 → 상세 펼치기
  - 모바일 스와이프 탭 (재물 / 직업 / 관계)

- [ ] **부적 카드 이미지 저장/공유**
  - 최종 판결문을 시각적 카드(Canvas/SVG→PNG)로 변환
  - 공유 버튼 (카카오톡, 인스타 스토리, 링크 복사)
  - 채팅 히스토리는 세션 종료 시 자동 삭제

### Nice-to-Have
- [ ] 로딩 시 3신 격론 스트리밍 연출
- [ ] Lighthouse 모바일 90+ 최적화

## v1.1 — "관계의 확장과 구체적 해결책" (리텐션 강화)

> 목표: 핵심 타겟(20~35 여성)의 최대 관심사 커버

### Must-Have

- [ ] **궁합 3신 데이터 연동**
  - generateCompatibility()에 자미두수 데이터 추가
    - 부처궁(夫妻宮), 교우궁(交友宮) 별 배치
  - 서양점성 데이터 추가
    - 금성/화성 위치, 시나스트리(두 차트 겹침) 요소
  - 프롬프트: 3신이 각자 궁합을 분석 → 합의
  - CompatibilityReport에 3신 개별 견해 필드 추가

- [ ] **연애/인연운 3곡선 그래프 (loveGraph)**
  - TotalReport에 `loveGraph: GraphPeriod[]` 추가
  - AI 프롬프트 규칙: 연애운 전용 3점수 산출 기준
    - sajuScore: 도화살, 홍염살, 천을귀인 기반
    - ziweiScore: 부처궁 화록/화기 기반
    - natalScore: 금성 트랜짓 + 7하우스 기반
  - report/page.tsx에 loveGraph 렌더링 섹션 추가

- [ ] **쿠키 콤보 결제 유도**
  - "3신 모두 열람 → Verdict 무료" 로직
  - 할인 번들 UI

### Nice-to-Have
- [ ] 궁합 바이럴: 초대 URL → 카카오톡 OG 이미지
- [ ] 캐릭터 일러스트 (AI 생성 or 커미션)

## v2.0 — "일상으로의 침투" (습관 형성 & 수익화 고도화)

> 목표: 일회성 앱 → 반복 방문 서비스

### Must-Have

- [ ] **올해의 삼신 예언**
  - 12개월 × (sajuScore + ziweiScore + natalScore)
  - 월별 핵심 메시지 + phaseType
  - 세운(년운) + 월운 데이터 필요 (사주)
  - 유년반(流年盤) 계산 필요 (자미두수)
  - 트랜짓 계산 (@orrery/core 지원 확인)

- [ ] **부적 콜렉션 앨범**
  - 과거 판결문들을 모아보는 갤러리
  - 날짜별, 주제별 필터

- [ ] **데일리 예언 (줌인)**
  - 오늘의 삼신 한마디
  - Push 알림 연동 (PWA)

### Nice-to-Have
- [ ] 유료 심층 질문권 (채팅 프리미엄)
- [ ] 캐릭터 굿즈/부적 IP 확장
- [ ] SEO: 별자리별, 띠별 랜딩 페이지
